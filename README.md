## Term Tetris
Play tetris in your terminal

## Requirements
java 1.8
Ascii compatible terminal

## How to play
In terminal:
java -jar TermTetris.jar

1. press `r` to rotate shape
2. press `space` to drop shape right away
3. press `<-` to move shape left
4. press `->` to move shape right
